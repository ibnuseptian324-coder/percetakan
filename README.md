# Website Percetakan Nadien's Grafika

Website HTML untuk **Nadien's Grafika - Percetakan & Digital Printing**.

## Halaman

- `index.html` - halaman utama, layanan, produk, dan form pesanan pelanggan.
- `login.html` - halaman login admin.
- `admin.html` - dashboard admin untuk data pelanggan dan kelola produk.

## Login Admin

```txt
Username: admin
Password: nadien123
```

## Fitur

- Halaman utama percetakan.
- Produk tampil di halaman utama.
- Form data pelanggan.
- Data pelanggan masuk ke admin.
- Admin bisa ubah status pesanan.
- Admin bisa hapus pesanan.
- Admin bisa tambah produk.
- Admin bisa edit produk.
- Admin bisa hapus produk.
- Produk bisa aktif/nonaktif.

## Cara Pakai

1. Buka `index.html`.
2. Isi form pesanan di bagian **Pesan Cetak**.
3. Klik **Login Admin**.
4. Masuk dengan akun admin.
5. Kelola data pelanggan dan produk dari `admin.html`.

## Catatan Penting

Versi ini memakai HTML, CSS, dan JavaScript murni.

Data disimpan menggunakan `localStorage` browser, jadi:

- Data hanya tersimpan di browser/perangkat yang sama.
- Data belum tersimpan online ke database.
- Jika cache/browser dibersihkan, data bisa hilang.

Untuk website bisnis asli yang datanya aman dan bisa dibuka dari banyak perangkat, gunakan backend seperti PHP + MySQL.
